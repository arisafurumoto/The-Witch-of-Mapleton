# Approved Character Style Pack

This directory contains the approved original Mapleton humanoid style references:

```text
mapleton_humanoid_south.png  Proportion, pose, perspective, and baseline authority
mapleton_head_south.png      Face scale and feature-placement authority
mapleton_palette.png         Outline, skin, neutral cloth, and shading authority
```

Do not place borrowed game assets or rejected Mapleton generations here. Temporary
study references stay outside the repository. Follow
`docs/PIXEL_CHARACTER_GENERATION_WORKFLOW.md`.

Approved files:

```text
mapleton_humanoid_south.png        Native 96x112 mannequin
mapleton_humanoid_south.face.json  Shared-face and eye validation metadata
mapleton_head_south.png            Native head construction reference
mapleton_palette.png               Approved 48-color palette reference
mapleton_eye_templates.json        Reusable native eye-role geometry
```

The preserved high-resolution generation source lives outside runtime art at
`concept_art/style_reference/mapleton_humanoid_south_source.png`.

The approved neutral eye grid is `UUU / WPP / WPP` for the left eye and its
horizontal mirror for the right. The south pose was verified from complete directional
reference sheets rather than a south-west-only NPC pose.
